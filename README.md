## Overview
EduAI is a multimodal chatbot built using:
- Groq LLM API
- OCR-based image understanding (Tesseract.js)
- Retrieval-Augmented Generation (RAG)
- Vercel deployment

## Features
- Text-based academic Q&A
- Image upload support
- Roadmap and diagram explanation
- Context-aware responses using RAG

## Tech Stack
- Node.js
- Groq API
- Tesseract.js
- Vercel

## Setup

1. Clone the repo
2. Run: npm install
3. Create .env.local
4. Add:
   GROQ_API_KEY=your_key_here
5. Run:
   npx vercel dev
